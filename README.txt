animal.py - основной файл с итоговой анимацией
spine.py - классы, обеспечивающие расчет и отрисовку позвоночника (удобно использовать для отладки модификаций)

gui.py - базовый класс для любого шейдера
colors.py - цвета и палитры
sdf.py - некоторый функции для расчета поля расстояний
core.py - вспомогательные функции, например hash
